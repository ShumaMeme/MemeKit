## 项目简介

MemeKit 是一款 Android 通用向玩机工具箱，依托 Android 底层 ADB/Fastboot 机制实现刷机、ROOT、设备调试、分区管理等全套功能。界面采用 Fluent Design 现代化设计，原生适配 Windows 美学。

## 2. 核心优势

### 全设备兼容
- **品牌全覆盖**：小米、拯救者、摩托罗拉、联想、一加、三星、真我等
- **芯片适配**：高通、联发科、展锐、三星 SoC
- **设备类型**：手机、平板、电视盒子、智能手表
- **固件兼容**：官方原厂包、第三方 ROM、救砖固件

### 安全防护机制
- 自动识别 ADB / Fastboot / FastbootD / EDL / Rec 设备模式
- 模式权限隔离，杜绝误刷分区
- 自动识别 A/B 双分区机型，自动切换槽位
- 完整操作日志留存，方便故障排查

### 流畅现代化界面
- 深浅双主题，跟随系统自动切换
- 全异步多线程，操作不卡顿 UI
- 原生 Windows 云母/毛玻璃特效
- 启动动画、弹窗模糊美化

## 3. 全部功能

| 功能模块 | 说明 |
|---------|------|
| **仪表盘** | 读取设备序列号、硬件参数、电池状态、系统版本、锁机状态；一键切换启动模式，重装 ADB 驱动、打开终端 |
| **一键ROOT** | 兼容 Magisk / KernelSU / APatch / Magisk Alpha / Sukisu Ultra 等方案；自动修补 boot、init_boot 镜像并刷入 |
| **刷机中心** | 支持散包批量刷写、小米线刷脚本、ADB Sideload、Payload.bin 固件提取、单分区自定义刷入 |
| **快捷指令** | 内置常用 ADB、Fastboot、CMD 调试命令，支持自定义新增、分类管理，一键执行并实时输出日志 |
| **投屏控制** | 内置 Scrcpy，USB 低延迟投屏，可直接操控手机，刷机调试同步进行 |
| **字库分区备份&还原** | 完整备份字库分区；批量恢复字库分区镜像，实时读写进度反馈 |
| **软件&文件管理** | 批量安装/导出 APK、卸载冻结应用；设备与电脑双向批量传输文件，直接浏览系统分区 |
| **设置面板** | 切换主题、检测依赖工具（ADB/7z等）、检查更新、查看项目信息 |

## 4. 操作教程

### 初次使用
1. 手机开启「开发者选项」→ 启用「USB 调试」
2. 数据线连接电脑，手机上授权设备调试
3. 软件自动识别设备，即可使用全部功能

### 基础刷机流程
1. 打开「刷机中心」，选择对应刷机模式
2. 载入固件文件夹 / 镜像文件
3. 确认设备当前启动模式无误
4. 点击开始，等待刷写完成

### ROOT 操作流程
1. 进入「一键 ROOT」，选择 Root 管理器方案
2. 工具自动推送 ROOT 管理器至手机
3. 根据弹窗提示完成修补与刷入

## 5. 风险须知

> ⚠️ **重要提示**：请务必仔细阅读以下内容

1. 刷机、解锁 Bootloader 会清空手机全部数据，操作前务必备份资料
2. 仅可使用对应机型固件，错刷镜像会导致设备变砖
3. 部分厂商设备解锁 BL 会失去官方保修
4. 本工具仅用于个人学习研究，操作产生设备损坏由使用者自行承担
