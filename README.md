MemeKit Android玩机工具箱
 
MemeKit 基于 GitHub 项目 Tobatools（拖把工具箱）二次开发衍生，上游项目源码地址：https://github.com/Tobapuww/Tobatools
 
项目简介
 
MemeKit 是一款通用安卓刷机工具箱，适配市面绝大多数安卓设备，不绑定单一品牌，依托 Android 底层 ADB/Fastboot 机制实现刷机、ROOT、设备调试、分区管理等全套功能，界面采用 Fluent Design 现代化设计，Windows 平台原生适配。

核心优势
 
1. 全设备兼容
 
- 品牌全覆盖：小米、拯救者、摩托罗拉、联想、一加、三星、真我等
- 芯片适配：高通、联发科、展锐、三星SoC
- 设备类型：手机、平板、电视盒子、智能手表
- 固件兼容：官方原厂包、第三方ROM、救砖固件
 
2. 安全防护机制
 
- 自动识别 ADB / Fastboot / FastbootD / EDL / Rec 设备模式
- 模式权限隔离，杜绝误刷分区
- 自动识别A/B双分区机型，自动切换槽位
- 完整操作日志留存，方便故障排查
 
3. 流畅现代化界面
 
- 深浅双主题，跟随系统自动切换
- 全异步多线程，刷机操作不卡顿UI
- 原生Windows云母/毛玻璃特效
- 启动动画、弹窗模糊美化
 
 
 
全部功能
 
仪表盘（设备总览）
 
读取设备序列号、硬件参数、电池状态、系统版本、锁机状态；支持一键切换设备启动模式，快速重装ADB驱动、打开终端。
 
一键ROOT
 
兼容 Magisk / KernelSU / APatch / Magisk Alpha /Sukisu Ultra等多种ROOT方案；支持一键自动修补boot、init_boot镜像并刷入手机
 
刷机中心
 
支持散包批量刷写、小米线刷脚本、ADB Sideload、Payload.bin固件提取、单分区自定义刷入。
 
快捷指令（Adb、Fastboot、CMD）
 
内置常用调试命令，支持自定义新增、分类管理，一键执行并实时输出日志。
 
投屏控制
 
内置Scrcpy，USB低延迟投屏，可直接在电脑操控手机，刷机调试同步进行。
 
字库分区备份&还原
 
完整备份字库分区；支持批量恢复镜像，实时显示读写进度。
 
杂项高级工具
 
1. 固件处理：Payload解析、OPPO OFP固件解密提取
2. 设备解锁：一键BL解锁、FRP锁清除（需Root）
3. 模块管理：Magisk/KSU模块批量安装、禁用、备份
4. 系统参数修改：分辨率、DPI、动画速度、电量模拟、隐藏状态栏图标
5. 内置Shell终端，可运行自定义脚本
 
软件&文件管理
 
批量安装/导出APK、卸载冻结应用；设备与电脑双向批量传输文件，直接浏览系统分区。
 
设置面板
 
切换主题、检测依赖工具（ADB/7z等）、在线检查更新、查看项目信息。
 
 
 
 
初次使用步骤
 
1. 手机开启开发者选项、USB调试
2. 数据线连接电脑，手机授权设备调试
3. 软件自动识别设备，即可使用全部功能
 
 
 
操作教程
 
基础刷机流程
 
1. 打开「刷机中心」，选择对应刷机模式
2. 载入固件文件夹/镜像文件
3. 确认设备当前启动模式无误
4. 点击开始，等待刷写完成
 
ROOT操作流程
 
1. 进入「一键ROOT」，选择适配方案
2. 工具自动推送ROOT管理器至手机
3. 根据软件弹窗提示完成修补与刷入

风险须知
 
1. 刷机、解锁Bootloader会清空手机全部数据，操作前务必备份资料
2. 仅可使用对应机型固件，错刷镜像会导致设备变砖
3. 部分厂商设备解锁BL会失去官方保修
4. 本工具仅用于个人学习研究，操作产生设备损坏由使用者自行承担
